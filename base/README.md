## Image Description
This image is the basis of the image to do barebone Go development. It does not setup any editors for Go.

## Image Contents
It includes:
- A non-root user named `gopher`
- Go 1.5.2
- Go Tools:
  - Gocode
  - Godef
  - Golint
  - Go Find References
  - Go Returns
  - GoRename
  - Delve
